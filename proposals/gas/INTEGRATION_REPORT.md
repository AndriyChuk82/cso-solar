### 3. Додано API endpoints для Proposals

**GET endpoints:**
- `getAllProducts` - завантаження продуктів з зовнішніх таблиць
- `getCustomMaterials` - завантаження власних матеріалів
- `getRates` - отримання курсів валют з API Goverla
- `getAllData` - завантаження всіх даних одним запитом

**POST endpoints:**
- `updateMaterialPrice` - оновлення ціни матеріалу
- `addCustomMaterial` - додавання власного матеріалу
- `deleteCustomMaterial` - видалення (деактивація) матеріалу
